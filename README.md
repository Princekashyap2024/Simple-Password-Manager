        <h1 style="color: #2c3e50; text-align: center; margin-bottom: 10px;">Simple Password Manager</h1>
        <p style="text-align: center; font-style: italic; margin-bottom: 20px;"><strong>Simple Password Manager</strong> is a basic browser extension that allows you to store, manage, and generate strong passwords conveniently. This tool is designed for personal use and helps you keep track of your credentials securely within your browser.</p>
        <hr style="border: none; border-top: 2px solid #ddd; margin: 20px 0;">
        <h2 style="color: #2980b9;">Features</h2>
        <ul style="padding-left: 20px;">
            <li><strong>Password Storage:</strong> Save and manage your credentials (website, username, and password) easily.</li>
            <li><strong>Password Generation:</strong> Generate strong random passwords with the click of a button.</li>
            <li><strong>User-Friendly Interface:</strong> Simple and intuitive design for ease of use.</li>
            <li><strong>Local Storage:</strong> All passwords are saved in the browser's local storage for privacy.</li>
        </ul>
        <hr style="border: none; border-top: 2px solid #ddd; margin: 20px 0;">
        <h2 style="color: #2980b9;">How It Works</h2>
        <ol style="padding-left: 20px;">
            <li><strong>Generate Passwords:</strong> Use the "Generate" button to create a secure random password.</li>
            <li><strong>Save Passwords:</strong> Enter the website, username, and password details to save them securely.</li>
            <li><strong>Retrieve Saved Passwords:</strong> View a list of all your saved passwords anytime from the extension.</li>
        </ol>
        <hr style="border: none; border-top: 2px solid #ddd; margin: 20px 0;">
        <h2 style="color: #2980b9;">Technologies Used</h2>
        <ul style="padding-left: 20px;">
            <li><strong>JavaScript:</strong> Handles the logic for storing and retrieving passwords.</li>
            <li><strong>HTML/CSS:</strong> Provides the structure and styling for the extension.</li>
            <li><strong>Chrome Storage API:</strong> Utilizes browser storage to save passwords locally.</li>
        </ul>
        <hr style="border: none; border-top: 2px solid #ddd; margin: 20px 0;">
        <h2 style="color: #2980b9;">Installation</h2>
        <ol style="padding-left: 20px;">
            <li>Clone this repository to your local machine:</li>
            <pre style="background: #ecf0f1; padding: 10px; border-radius: 5px; margin: 10px 0; overflow: auto;">git clone https://github.com/&lt;your-username&gt;/simple-password-manager.git
cd simple-password-manager</pre>
            <li>Open your Chrome browser and navigate to <code style="background: #ecf0f1; padding: 2px 5px; border-radius: 3px;">chrome://extensions</code>.</li>
            <li>Enable <strong>Developer Mode</strong> by toggling the switch in the top right corner.</li>
            <li>Click on the <strong>Load Unpacked</strong> button and select the project folder.</li>
            <li>The extension will now appear in your browser toolbar.</li>
        </ol>
        <hr style="border: none; border-top: 2px solid #ddd; margin: 20px 0;">
        <h2 style="color: #2980b9;">Usage</h2>
        <ol style="padding-left: 20px;">
            <li>Click on the extension icon in your browser toolbar.</li>
            <li>Fill in the required fields (website, username, and password) to save credentials.</li>
            <li>Use the "Generate" button to create strong passwords for your accounts.</li>
            <li>View the saved passwords directly within the extension.</li>
        </ol>
        <hr style="border: none; border-top: 2px solid #ddd; margin: 20px 0;">
        <h2 style="color: #2980b9;">Limitations</h2>
        <p style="margin-bottom: 10px;">Passwords are stored in plain text in local storage. This implementation is not secure for sensitive data. Avoid using it for critical accounts unless improved with encryption.</p>
